在 Ubuntu 中安装 X11 可以通过以下步骤完成：

  1. 打开终端，输入以下命令更新软件包列表：

    sudo apt-get update

  2. 安装 X11 的核心组件：

    sudo apt-get install xorg

  3. 安装 X11 的开发工具：

    sudo apt-get install x11proto-dev x11-utils libx11-dev libxext-dev

  4. 重新启动 X11：

    sudo /etc/init.d/gdm3 restart

现在，X11 已经安装并启动了。你可以使用 X11 应用程序了。
