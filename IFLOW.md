# 项目概述

这是一个个人知识管理和资源收集的项目，主要包含以下几类内容：

1. **技术资源导航** (`index.html`): 这是项目的主页，提供大量技术学习资源、工具、文档和社区的链接，涵盖 AI、编程语言 (Python, C++, Qt)、开发工具、Linux、数据库、量化交易等多个领域。

2. **开发资源** (`dev.html`): 专门收集了 C++、Python、Java、QT 等编程语言和技术的开发资源，包括开源库、框架、教程视频、书籍和项目实战代码。

3. **软件资源** (`soft.html`): 提供各种 Windows 软件的下载链接和信息，包括操作系统镜像、Office 套件、开发工具 (IDE, 编辑器)、效率工具、录屏截图软件、系统优化工具等，并按时间进行了分类。

4. **运维资源** (`op.html`): 包含 Linux 系统 (CentOS7, Debian12) 配置、MySQL 数据库配置、常用服务器设置等运维相关的教程和文档链接。

5. **服务与工具** (`service.html`): 提供一些在线服务和工具的链接，例如在线图片编辑器 Photopea、天气预报等。

6. **期货交易** (`futures/`): 包含与期货交易相关的文档、书籍和策略资料。

7. **其他个人内容**: 项目中还包含一些个人博客、家庭相关页面 (`my/`)、澳门相关资料 (`macau/`) 等。

# 使用说明

这个项目本质上是一个静态的 HTML 网站和个人知识库。要浏览内容，只需在浏览器中打开 `index.html` 文件即可。`index.html` 页面提供了到其他主要页面 (`dev.html`, `soft.html`, `op.html`, `service.html`) 的导航链接。

- **浏览资源**: 打开 `index.html` 查看所有分类的技术资源链接。
- **查看开发资源**: 打开 `dev.html` 查看编程语言和开发工具相关的资源。
- **查找软件**: 打开 `soft.html` 查找和下载各种软件。
- **学习运维**: 打开 `op.html` 访问 Linux 和数据库相关的运维教程。
- **使用在线工具**: 打开 `service.html` 访问在线服务。

# 开发约定

由于这是一个主要用于信息收集和展示的静态网站项目，没有复杂的构建或运行流程。主要的开发约定体现在 HTML 和 CSS 的编写上：

- **HTML 结构**: 使用语义化的 HTML 标签来组织内容。
- **CSS 样式**: 使用内联样式和简单的 CSS 规则来美化页面。
- **链接管理**: 所有外部链接都使用 `target="_blank"` 在新标签页中打开。
- **本地资源**: 一些 CSS 和 JS 库 (如 jQuery, Bootstrap) 被下载到本地 `static/` 和 `js/` 目录中引用。

# 关键文件说明

- `index.html`: 项目的入口和主导航页面。
- `dev.html`: 开发者资源集合。
- `soft.html`: 软件下载和信息页面。
- `op.html`: 运维技术文档和教程链接。
- `service.html`: 在线工具和服务链接。
- `futures/index.md`: 期货交易相关书籍列表。
